\# Project Management Tool



A full-stack project management application built with Node.js, Express.js, MongoDB, HTML, CSS and JavaScript.



\## Features

\- User Registration and Login

\- Create Projects

\- Add Tasks

\- Task Status Management (To Do, In Progress, Done)

\- Comments on Tasks

\- Authentication using JWT

\- MongoDB Database Integration



\## Tech Stack

\- Frontend: HTML, CSS, JavaScript

\- Backend: Node.js, Express.js

\- Database: MongoDB

\- Authentication: JWT



\## Run Locally



```bash

npm install

node server.js

```



Open:



http://localhost:5000/dashboard.html
